# Guía para Beta Testers - TennisApp 🎾

¡Bienvenido al programa beta de TennisApp! Tu participación es fundamental para mejorar la aplicación.

## 🎯 Objetivo del Beta

Estamos probando las funcionalidades core de gestión de alumnos, calendario y sesiones. El análisis de video con IA vendrá más adelante.

## 📱 Funcionalidades Disponibles

### 1. Gestión de Usuarios
- **Agregar alumnos**: Crear perfiles con nombre, nivel, mano dominante, etc.
- **Editar información**: Actualizar datos de tus alumnos
- **Archivar**: Ocultar alumnos inactivos sin eliminarlos
- **Buscar**: Encontrar rápidamente usando la barra de búsqueda

### 2. Calendario y Sesiones
- **Programar clases**: Selecciona fecha, hora, duración y alumnos
- **Ver calendario**: Visualiza todas tus clases programadas
- **Editar sesiones**: Modifica horarios o información de clases
- **Detección de conflictos**: La app te avisa si intentas programar dos clases al mismo tiempo

### 3. Ubicaciones
- **Agregar clubes/canchas**: Registra dónde das clases
- **Gestionar información**: Dirección, notas, detalles de cada ubicación

### 4. Colaboradores (Solo Coaches)
- **Agregar staff**: Registra otros profesores o colaboradores
- **Asignar roles**: Define permisos de cada colaborador

## 🐛 Cómo Reportar Problemas

### Botón de Feedback
1. Busca el ícono de **chat** (💬) en la esquina superior izquierda junto al badge "BETA"
2. Toca el botón para abrir el formulario de feedback
3. Selecciona el tipo:
   - **Bug**: Algo no funciona correctamente
   - **Sugerencia**: Ideas para mejorar la app
   - **Pregunta**: Dudas sobre cómo usar algo
4. Describe el problema en detalle
5. Envía

### Qué Incluir en un Reporte de Bug
- **¿Qué intentabas hacer?** Ej: "Estaba agregando un nuevo alumno"
- **¿Qué pasó?** Ej: "La app se cerró cuando toqué Guardar"
- **¿Cómo reproducirlo?** Los pasos exactos para que ocurra de nuevo

### Qué Incluir en una Sugerencia
- **¿Qué te gustaría?** Sé específico
- **¿Por qué sería útil?** Explica el caso de uso
- **¿Cómo lo imaginas?** Describe cómo funcionaría

## 💡 Casos de Uso Principales

### Flujo Típico de un Coach
1. **Inicio**: Agregar tus ubicaciones (clubes/canchas donde das clases)
2. **Alumnos**: Registrar a todos tus alumnos actuales
3. **Calendario**: Programar las clases de la semana
4. **Gestión**: Ver el calendario, modificar horarios según necesidad

### Escenarios a Probar
- ✅ Crear alumnos con diferentes niveles
- ✅ Programar clase individual
- ✅ Programar clase grupal (varios alumnos)
- ✅ Intentar programar dos clases al mismo tiempo (debe avisar conflicto)
- ✅ Editar información de un alumno
- ✅ Cancelar/eliminar una sesión
- ✅ Buscar alumnos por nombre
- ✅ Archivar un alumno inactivo

## ❓ Preguntas Frecuentes (FAQ)

### ¿Puedo eliminar datos de prueba?
Sí, puedes eliminar alumnos, sesiones y ubicaciones. Durante el beta, tus datos son solo para pruebas.

### ¿Qué pasa con mis datos después del beta?
Te avisaremos antes de cualquier cambio. Tendrás la opción de mantenerlos o empezar limpio en la versión final.

### ¿Puedo invitar a otros coaches?
Por ahora estamos en beta cerrada. Pronto abriremos invitaciones.

### No encuentro una función
Si necesitas algo que no está disponible, ¡envíanos una sugerencia! Muchas funciones están planificadas pero queremos saber qué priorizas.

### ¿La app funciona sin internet?
Actualmente necesitas conexión para usar la app. El modo offline llegará en futuras versiones.
